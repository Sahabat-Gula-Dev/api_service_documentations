Carousel Service

Endpoint-endpoint ini digunakan untuk mengelola gambar-gambar yang akan ditampilkan pada carousel di halaman utama aplikasi.

Carousels

POST /carousels

Tambah Carousel Baru

Menambahkan gambar carousel baru. Endpoint ini memerlukan otorisasi dan mengirim data sebagai multipart/form-data.

Authorization: Bearer Token

Request Body multipart/form-data

Key Tipe Deskripsi
image_file file File gambar yang akan ditampilkan. Wajib diisi jika image_url tidak ada.
image_url string URL gambar eksternal. Wajib diisi jika image_file tidak ada.
target_url string URL tujuan saat gambar di-klik. Opsional.

Success Response 201 Created

{
    "status": "success",
    "message": "Carousel berhasil ditambahkan.",
    "data": {
        "carousel": {
            "id": 1,
            "image_url": "https://url.supabase.co/storage/v1/object/public/carousels/image.jpg",
            "target_url": "https://sahabatgula.com/artikel/gaya-hidup-sehat",
            "created_at": "2025-09-20T17:00:00.000Z"
        }
    }
}
GET /carousels

Daftar Carousel

Mengambil daftar gambar carousel dengan dukungan pagination dan sorting.

Authorization: None

Query Parameters

Key Tipe Deskripsi
page integer Nomor halaman. Default: 1. Opsional.
limit integer Jumlah item per halaman. Default: 20. Opsional.
sort string Urutan pengurutan. Contoh: created_at.asc. Default: created_at.desc. Opsional.

Success Response 200 OK

{
    "status": "success",
    "data": [
        {
            "id": 1,
            "image_url": "https://url.supabase.co/storage/v1/object/public/carousels/image1.jpg",
            "target_url": "https://sahabatgula.com/artikel/gaya-hidup-sehat",
            "created_at": "2025-09-20T17:00:00.000Z"
        },
        {
            "id": 2,
            "image_url": "https://url.supabase.co/storage/v1/object/public/carousels/image2.jpg",
            "target_url": "https://sahabatgula.com/event/webinar-diabetes",
            "created_at": "2025-09-19T11:00:00.000Z"
        }
    ],
    "meta": { "page": 1, "limit": 20, "total": 5 }
}
GET /carousels/{id}

Detail Carousel

Mengambil detail satu data carousel berdasarkan ID-nya.

Authorization: None

Success Response 200 OK

{
    "status": "success",
    "data": {
        "carousel": {
            "id": 1,
            "image_url": "https://url.supabase.co/storage/v1/object/public/carousels/image1.jpg",
            "target_url": "https://sahabatgula.com/artikel/gaya-hidup-sehat",
            "created_at": "2025-09-20T17:00:00.000Z"
        }
    }
}
PUT /carousels/{id}

Perbarui Carousel

Memperbarui data carousel berdasarkan ID. Semua field bersifat opsional (kirim hanya yang ingin diubah).

Authorization: Bearer Token

Request Body multipart/form-data

Sama seperti request body untuk POST /carousels, tetapi semua field bersifat opsional.

Success Response 200 OK

{
    "status": "success",
    "message": "Carousel berhasil diperbarui.",
    "data": {
        "carousel": {
            "id": 1,
            "image_url": "https://url.supabase.co/storage/v1/object/public/carousels/new-image.jpg",
            "target_url": "https://sahabatgula.com/promo/diskon-spesial",
            "created_at": "2025-09-20T17:00:00.000Z"
        }
    }
}
DELETE /carousels/{id}

Hapus Carousel

Menghapus data carousel secara permanen berdasarkan ID.

Authorization: Bearer Token

Success Response 200 OK

{
    "status": "success",
    "message": "Carousel berhasil dihapus."
}
