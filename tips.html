Tips Service

Endpoint-endpoint ini digunakan untuk mengelola data tips singkat yang dapat ditampilkan secara acak di aplikasi.

Tips

POST /tips

Tambah Tips Baru

Menambahkan data tips baru ke dalam database. Endpoint ini memerlukan otorisasi. Minimal salah satu dari heading atau content harus diisi.

Authorization: Bearer Token

Request Body application/json

Key Tipe Deskripsi
heading string Judul atau kalimat pembuka tips. Opsional.
content string Isi konten dari tips. Opsional.

Success Response 201 Created

{
    "status": "success",
    "message": "Tips berhasil ditambahkan.",
    "data": {
        "tip": {
            "id": 1,
            "heading": "Tips Hidrasi",
            "content": "Pastikan minum 8 gelas air setiap hari untuk membantu fungsi tubuh yang optimal.",
            "created_at": "2025-09-20T18:00:00.000Z"
        }
    }
}
GET /tips

Ambil Tips Acak

Mengambil satu data tips secara acak dari database. Endpoint ini bersifat publik.

Authorization: None

Success Response 200 OK

{
    "status": "success",
    "data": {
        "tip": {
            "id": 1,
            "heading": "Tips Hidrasi",
            "content": "Pastikan minum 8 gelas air setiap hari untuk membantu fungsi tubuh yang optimal.",
            "created_at": "2025-09-20T18:00:00.000Z"
        }
    }
}
GET /tips/{id}

Detail Tips

Mengambil detail satu data tips berdasarkan ID-nya.

Authorization: None

Success Response 200 OK

{
    "status": "success",
    "data": {
        "tip": {
            "id": 1,
            "heading": "Tips Hidrasi",
            "content": "Pastikan minum 8 gelas air setiap hari untuk membantu fungsi tubuh yang optimal.",
            "created_at": "2025-09-20T18:00:00.000Z"
        }
    }
}
PUT /tips/{id}

Perbarui Tips

Memperbarui data tips berdasarkan ID. Semua field bersifat opsional (kirim hanya yang ingin diubah).

Authorization: Bearer Token

Request Body application/json

Key Tipe Deskripsi
heading string Judul baru. Opsional.
content string Konten baru. Opsional.

Success Response 200 OK

{
    "status": "success",
    "message": "Tips berhasil diperbarui.",
    "data": {
        "tip": {
            "id": 1,
            "heading": "Tips Tidur Berkualitas",
            "content": "Hindari layar gadget 1 jam sebelum tidur untuk kualitas istirahat yang lebih baik.",
            "created_at": "2025-09-20T18:00:00.000Z"
        }
    }
}
DELETE /tips/{id}

Hapus Tips

Menghapus data tips secara permanen berdasarkan ID.

Authorization: Bearer Token

Success Response 200 OK

{
    "status": "success",
    "message": "Tips berhasil dihapus."
}
