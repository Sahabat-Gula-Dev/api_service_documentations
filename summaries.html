Summary Service

Endpoint-endpoint ini digunakan untuk mengambil data ringkasan (summary) aktivitas dan konsumsi pengguna dalam rentang waktu tertentu. Semua endpoint di sini memerlukan otorisasi.

Summaries

GET /daily-summary

Ringkasan Konsumsi Hari Ini

Mengambil data ringkasan total nutrisi, kalori terbakar, langkah, dan air yang dikonsumsi untuk hari ini (sesuai zona waktu server).

Authorization: Bearer Token

Success Response 200 OK

Mengembalikan satu objek ringkasan untuk hari ini.

{
    "status": "success",
    "data": {
        "summary": {
            "date": "2025-09-20",
            "nutrients": {
                "calories": 1850.5,
                "carbs": 210.2,
                "protein": 80.5,
                "fat": 60.1,
                "sugar": 45.0,
                "sodium": 1.5,
                "fiber": 25.5,
                "potassium": 2.1
            },
            "activities": {
                "burned": 350
            },
            "steps": 8500,
            "water": 2000
        }
    }
}
