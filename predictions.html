Prediction Service

Endpoint ini berfungsi sebagai jembatan antara aplikasi dan model Machine Learning. Pengguna dapat mengunggah gambar makanan, dan server akan mengembalikan nama prediksi makanan tersebut beserta daftar makanan serupa yang ada di database.

Food Prediction

POST /predictions

Prediksi Gambar Makanan

Mengunggah sebuah file gambar untuk diproses oleh model Machine Learning. Endpoint ini akan mengembalikan nama makanan yang diprediksi, URL gambar yang telah diunggah, dan daftar makanan dari database yang cocok dengan nama prediksi.

Authorization: None

Request Body multipart/form-data

Key Tipe Deskripsi
image file File gambar makanan yang akan diprediksi. Ukuran maksimal 5MB. Wajib.

Success Response 200 OK

Respons berisi nama prediksi dan daftar makanan terkait.
