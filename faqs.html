FAQ Service

Endpoint-endpoint ini digunakan untuk mengelola semua data yang berkaitan dengan Pertanyaan yang Sering Diajukan (FAQ), termasuk manajemen kategori FAQ.

FAQs

POST /faqs

Tambah FAQ Baru

Menambahkan data FAQ baru ke dalam database. Endpoint ini memerlukan otorisasi.

Authorization: Bearer Token

Request Body application/json

Key Tipe Deskripsi
question string Teks pertanyaan. Wajib.
answer string Teks jawaban. Wajib.
category_id integer ID kategori yang sudah ada. Opsional, salah satu dari category_id atau category_name harus ada.
category_name string Nama kategori baru yang akan dibuat. Opsional.

Success Response 201 Created

{
    "status": "success",
    "message": "FAQ berhasil ditambahkan.",
    "data": {
        "faq": {
            "id": 1,
            "question": "Bagaimana cara mendaftar?",
            "answer": "Anda bisa mendaftar melalui halaman registrasi dengan mengisi email dan password.",
            "created_at": "2025-09-20T18:30:00.000Z",
            "category_id": 1
        }
    }
}
GET /faqs

Daftar FAQ

Mengambil daftar FAQ dengan dukungan pagination, pencarian (query `q`), dan filter.

Authorization: None

Query Parameters

Key Tipe Deskripsi
q string Teks pencarian untuk pertanyaan atau jawaban. Opsional.
category_id integer Filter berdasarkan ID kategori. Opsional.
category_name string Filter berdasarkan nama kategori. Opsional.
page integer Nomor halaman. Default: 1. Opsional.
limit integer Jumlah item per halaman. Default: 20. Opsional.
sort string Urutan pengurutan. Contoh: question.asc. Default: created_at.desc. Opsional.

Success Response 200 OK

{
    "status": "success",
    "data": [
        {
            "id": 1,
            "question": "Bagaimana cara mendaftar?",
            "answer": "Anda bisa mendaftar melalui halaman registrasi...",
            "created_at": "2025-09-20T18:30:00.000Z",
            "category_id": 1,
            "faq_categories": {
                "id": 1,
                "name": "Umum"
            }
        }
    ],
    "meta": { "page": 1, "limit": 20, "total": 10 }
}
GET /faqs/{id}

Detail FAQ

Mengambil detail satu data FAQ berdasarkan ID-nya.

Authorization: None

Success Response 200 OK

{
    "status": "success",
    "data": {
        "faq": {
            "id": 1,
            "question": "Bagaimana cara mendaftar?",
            "answer": "Anda bisa mendaftar melalui halaman registrasi dengan mengisi email dan password.",
            "created_at": "2025-09-20T18:30:00.000Z",
            "category_id": 1,
            "faq_categories": {
                "id": 1,
                "name": "Umum"
            }
        }
    }
}
PUT /faqs/{id}

Perbarui FAQ

Memperbarui data FAQ berdasarkan ID. Semua field bersifat opsional (kirim hanya yang ingin diubah).

Authorization: Bearer Token

Request Body application/json

Sama seperti request body untuk POST /faqs, tetapi semua field bersifat opsional.

Success Response 200 OK

{
    "status": "success",
    "message": "FAQ berhasil diperbarui."
}
DELETE /faqs/{id}

Hapus FAQ

Menghapus data FAQ secara permanen berdasarkan ID.

Authorization: Bearer Token

Success Response 200 OK

{
    "status": "success",
    "message": "FAQ berhasil dihapus."
}

FAQ Categories

GET /faq-categories

Daftar Kategori FAQ

Mengambil daftar semua kategori FAQ.

Authorization: None

Success Response 200 OK

{
    "status": "success",
    "data": [
        { "id": 1, "name": "Umum" },
        { "id": 2, "name": "Akun" }
    ]
}
POST /faq-categories

Buat Kategori FAQ Baru

Menambahkan kategori FAQ baru.

Authorization: Bearer Token

Request Body application/json

Key Tipe Deskripsi
name string Nama kategori baru. Wajib.

Success Response 201 Created

{
    "status": "success",
    "message": "Kategori berhasil ditambahkan.",
    "data": {
        "category": { "id": 3, "name": "Pembayaran" }
    }
}
